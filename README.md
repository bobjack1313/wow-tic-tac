# WoW Tic Tac

A research simulator for the Midnight Imperator Averzian tic-tac-toe mechanic.

## Goals

- Reverse engineer the boss selection algorithm
- Test optimal raid strategies
- Determine whether Round 6 is achievable
- Evaluate movement-safe board states

## Features

- Manual boss mode
- Random opening generator
- Legal move validation
- King-adjacency rule engine
- Unit tests

## Run

python3 play.py
